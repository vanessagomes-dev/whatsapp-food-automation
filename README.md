# Automação de Mensagens WhatsApp (Protótipo)

Protótipo de automação desenvolvido em Python para envio de mensagens
programadas (imagem + texto) em grupos do WhatsApp.

## Funcionalidades
- Agendamento automático por horário
- Mensagens configuráveis
- Simulação de envio em terminal
- Arquitetura preparada para WhatsApp Business API

## Tecnologias
- Python 3
- APScheduler
- PEP8 / Flake8

## Execução local
```bash
python backend/app/main.py
